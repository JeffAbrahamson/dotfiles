# Mutt

This directory contains Mutt mail client configuration and a few helper scripts.

## Contents

* [`bin/`](bin/README.md) contains small helper programs used by the mail setup.
* [`mutt/`](mutt/README.md) contains the actual mail client config files.
* [`muttrc`](muttrc) is a top-level config entry file.
* [`install.sh`](install.sh) installs the Mutt config.
* [`test.sh`](test.sh) runs Mutt-specific checks.
