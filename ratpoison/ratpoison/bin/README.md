# Ratpoison Helper Scripts

This directory contains scripts used by the Ratpoison environment for status output, power monitoring, and quick utilities.

## Notable scripts

* [`ratpoison-status`](ratpoison-status), [`ratpoison-power-status`](ratpoison-power-status), and [`ratpoison-wifi-status`](ratpoison-wifi-status) expose machine state to the status bar.
* [`ratpoison-power-monitor`](ratpoison-power-monitor) and [`ratpoison-lid-monitor`](ratpoison-lid-monitor) react to power and lid events.
* [`ratpoison-gtd`](ratpoison-gtd), [`ratpoison-cal`](ratpoison-cal), and [`ratpoison-dates`](ratpoison-dates) provide quick launcher-style utilities inside the Ratpoison environment.
