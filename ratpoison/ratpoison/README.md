# Ratpoison Config

This directory contains the main Ratpoison configuration and supporting assets.

## Contents

* [`rc`](rc) is the main Ratpoison configuration.
* [`ratpoison.desktop`](ratpoison.desktop) is a desktop entry for launching Ratpoison.
* [`acpi-local-lid.sh`](acpi-local-lid.sh) handles lid-related power or display behavior.
* [`bin/`](bin/README.md) contains helper scripts used by the window-manager setup.
