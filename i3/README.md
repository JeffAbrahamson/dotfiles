# i3

This directory contains the configuration for the i3 window manager plus helper scripts it calls. The multiple host-specific config files suggest a shared base config with machine-local variations.

## Contents

* [`i3/`](i3/README.md) contains the main config files and helper scripts.
* [`install.sh`](install.sh) installs the i3 configuration.
* [`test.sh`](test.sh) runs i3-specific checks.
