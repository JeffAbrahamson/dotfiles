# i3 Helper Scripts

This directory contains scripts used by the i3 configuration for status display, monitor handling, power events, and workspace restoration.

## Notable scripts

* [`i3-env-stats`](i3-env-stats), [`i3-power-monitor`](i3-power-monitor), and [`i3-power-status`](i3-power-status) expose machine state to status bars and notifications.
* [`i3-lid-monitor`](i3-lid-monitor) and [`i3-xrandr`](i3-xrandr) help keep display behavior sane across docking and lid-close scenarios.
* [`i3-restore-2`](i3-restore-2), [`i3-restore-2b`](i3-restore-2b), [`i3-restore-jba`](i3-restore-jba), and [`i3-restore-jbi`](i3-restore-jbi) are workspace/session restoration helpers for specific environments.
