# i3 Config

This directory contains the main i3 configuration files and supporting desktop scripts.

## Contents

* [`config`](config), [`config_base`](config_base), and host-specific variants such as [`config.birdsong`](config.birdsong) and [`config.vogel`](config.vogel) define the window-manager setup.
* [`dunstrc`](dunstrc) and [`i3status.conf`](i3status.conf) configure notifications and status output.
* [`bin/`](bin/README.md) contains helper scripts invoked by keybindings or background services.
