# Notion Config

This directory contains the actual Notion window-manager configuration.

## Contents

* [`cfg_notion.lua`](cfg_notion.lua), [`cfg_notioncore.lua`](cfg_notioncore.lua), and [`cfg_tiling.lua`](cfg_tiling.lua) define the active configuration.
* [`examples/`](examples/README.md) contains sample or reference configuration files kept nearby for comparison.
