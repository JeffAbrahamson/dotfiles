# Sway

This directory contains the Sway window-manager setup plus supporting notes and helper scripts. It is one of the more involved desktop areas in the repo because it includes both active config and migration/design notes.

## Contents

* [`NOTES/`](NOTES/README.md) contains working notes about Sway setup decisions and open questions.
* [`bash/`](bash/README.md) contains shell integration for Sway sessions.
* [`sway/`](sway/README.md) contains the active Sway config and helper scripts.
* [`Makefile`](Makefile) provides the `install` and `test` targets for the Sway setup.
* [`sway-notes.txt`](sway-notes.txt) is an older consolidated note file.
