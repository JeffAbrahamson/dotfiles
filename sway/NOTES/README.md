# Sway Notes

This directory contains planning notes, experiments, and operational reminders for the Sway environment. It is useful when adjusting the desktop setup and trying to remember why a particular choice was made.

## Contents

* [`config.txt`](config.txt), [`general.txt`](general.txt), and [`misc.txt`](misc.txt) contain broad setup notes.
* [`battery-notes.txt`](battery-notes.txt), [`network.txt`](network.txt), [`sound.txt`](sound.txt), [`swaybar.txt`](swaybar.txt), and [`swaylock.txt`](swaylock.txt) cover specific subsystems.
* [`examples.txt`](examples.txt), [`greeter.txt`](greeter.txt), [`systemd.txt`](systemd.txt), [`terminal.txt`](terminal.txt), and [`weather.txt`](weather.txt) hold narrower experiments and references.
* [`TODO.txt`](TODO.txt) tracks remaining work.
